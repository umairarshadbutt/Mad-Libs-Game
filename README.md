## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
To run the project smoothly, you must have the following installed on your machine:

- Git
- A web browser.

### Setup
You need to clone the project by running `https://github.com/umairarshadbutt/Mad-Libs-Game.git` You'd then navigate to the location to which you cloned the project and run the `index.html` file on your browser.
